Web Banner Animation Project

The purpose of this project is to use my existing static “Lucky Lizard Web Design” logo, and add a tongue that sticks out, grabs a fly, and then retracts. The animated gif shown here does not support some features that work in Adobe Animate. For example, in Animate, when you click the banner it brings you to my other website, which was built with Wix. Additionally, upon mouseover the logo changes color and then reverts back upon mouseout. I have included my Adobe Animate file here so that you can see those features if you have Animate. I was careful to use a naming convention that would make sense to myself and others.
